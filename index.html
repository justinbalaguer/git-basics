SETUP

Step 1. Create GitHub account

  • Create your GitHub account.


Step 2. Download Git

Download Git for Windows , Mac , Linux

  • Install Git

Step 3. Setup your terminal

  • Open Git Bash or any other terminal and setup git with your GitHub username and email.

  • $ git config --global user.name "Your username"
$ git config --global user.email "your_email@example.com"

UPLOADING PROJECTS TO GITHUB

Step 1. Create a Repository
  • Create your Repository.
Step 2. Open Terminal
  • Go to your Git Bash or any other terminal.
  • Change directory to your projects folder. 
    $ cd ../../xampp/htdocs/<project_folder>
  • Initialize Git & add the remote/origin 
    $ git init
$ git remote add origin https://github.com/<your_username>/<your_repository>.git
  • Commit and push project to your repository. 
    $ git add .
$ git commit -m "<your_commit_message>"
$ git push -u origin master

NOTE: you can now go to your GitHub Repository and reload to see if your project has been uploaded/imported.

CLONING PROJECTS FROM GITHUB

Step 1. Get Remote Url

  • Go to the Repository and copy the URL from Clone or download button.

Step 2. Open Terminal
  • Change directory to your desired folder.
  • Clone the repository 
    $ git clone https://github.com/<your_name>/<repository_name>.git
Step 3. Modify Project & push changes
  • You can now modify/edit the project you cloned.
  • Pull from repository to check for changes 
    $ git pull origin master
  • Stage, commit and push the changes you made 
    $ git add .
$ git commit "<your_commit_message>"
$ git push -u origin master

NOTE: you can now go to your GitHub Repository and reload to see if your changes has been updated/pushed.

Also note that this cloning guide is for repositories you have Push Permission

If you don't have push permission, you can read about Forking, Branching, Squashing commits etc. and use Pull Requests for the changes you made.
